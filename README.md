# Ruby Plugin for DocPad
Adds support to [DocPad](https://docpad.org) for writing your templates using [Ruby](http://en.wikipedia.org/wiki/ERuby)

Convention:  `.anything.rb|ruby|erb`


## Install

```
npm install --save docpad-plugin-ruby
```
